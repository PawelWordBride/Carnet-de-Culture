2. Sciences Physiques

2.1 Chimie

  • Chimie organique : carbone, fonctions, réactions.
  • Chimie inorganique : sels, métaux, minéraux, acides/bases.
  • Chimie physique : équilibres, cinétique, thermodynamique chimique.
  • Biochimie : molécules du vivant, enzymes, métabolisme.

2.2 Physique

  • Mécanique : forces, mouvement, énergie.
  • Thermodynamique : chaleur, entropie, machines.
  • Électromagnétisme : champs, circuits, ondes.
  • Optique : lumière, lentilles, instruments.
  • Optique atmosphérique : halos, mirages, arcs, diffusion.
  • Physique quantique : particules, probabilités, dualité.
  • Relativité : espace-temps, gravitation, vitesses extrêmes.

2.3 Propriétés de l’eau

  • Glace : structure, densité, flottabilité.
  • Tension superficielle : cohésion, gouttes, insectes.
  • Capillarité : montée dans les tubes/sols, plantes.
  • Solubilité : ions, polarité, dissolution.
